# TopengGO
TopengGO merupakan game mobile platformer edukatif  yang mengenalkan budaya Topeng Malangan kepada siswa sekolah dasar kelas 4 sampai 6 melalui petualangan interaktif. Dengan konsep belajar sambil bermain, Topeng GO dirancang menjadi media pembelajaran budaya lokal yang seru, menantang, dan mudah diakses generasi muda.
