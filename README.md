# TopengGO
Tutorial bermain TopengGO
1. Klik Tap to Start untuk memulai game TopengGO.
2. Arahkan karakter (Si Zar) ke kanan,kiri, dan melompat 
3. Di dalam permaian cari artefak topeng untuk
   mendapat petunjuk jawaban soal.
4. Lanjutkan perjalanan sambil mengumpulkan CoinGO
   hingga menemukan penanda (pointers) untuk membuka soal.
5. Jawab Soal dengan benar untuk mendapat CoinGO tambahan.
6. Kembali ke Menu Utama.
7. Klik Kreasi Topeng untuk mulai mewarnai Topeng Malangan.
8. Tunjukkan kreativitasmu!
9. Kembali ke Menu Utama.
10. Klik Buku Koleksi untuk melihat koleksi Topeng Malagan.
11. Baca latar belakang dan sejarah singkat untuk
    menambah wawasanmu!
